
This project is used forQmote testing

It generate the following information through Qmote Maker's Module

Combination Code		UARTmsg
0x01								seq=[sequence number]
0x02								et=[elapsed time in milliseconds]	
0x03								NONE
0x04								NONE
0x05								NONE
0x06								NONE
0x07								TempC=[temperature]
0x08								RhT=[RhT]	
0x09								field1=[temperature]	(for ThingSpeak)
0x0A								field2=[RhT] (for ThingSpeak)
